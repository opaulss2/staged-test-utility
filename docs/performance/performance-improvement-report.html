Startup cost was halved by replacing the eager swut.library.diagnostic_library import (which pulled in CLR and one_dad.dll_helpers) with a lazy importlib.import_module call deferred to first use. The deferred startup self-check (root.after(0, ...)) further eliminates blocking work before the first Tk paint cycle.
